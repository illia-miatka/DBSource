DBSource Manual

To get started, you need to add connections.
Go to Connection -> Add.
You will see New Connection window. Please enter:
- Connection - name of connection;
- Path - where the scripts will be saved;
- ByFolders - if true scripts will be saved in different folders, depending on the type;
On the right side select the connection type and fill all settings for this type. Then press Save connection.

New connection window

Main window

(1 – Button Connect/Disconnect, 2 – Button Load All, 3 – Button Current Schema, 4 – Button Date Filter, 5 – Button Name Filter, 6 – Button Load)

To connect to the server please select connection name from drop-down list (9). Next, click Connect button (1).
In the Object Types field (10), you will see a list of server object types. Select the required types and click Load Objects List button (6).
In the Objects field you will see a list of objects. Select the required objects. To save the database scripts for selected objects, click the Start button (7). 
If you want to to stop progress, press Stop button (8).

 

Additionally, you can filter the list of objects by 
- schema - button Current Schema (3), to choose objects from current DB schema (only Oracle) ;
- date - button Date Filter (4), to choose objects modified from selected date. To clear filter press Cancel;

 

- name - button Name Filter (5), to choose objects by name or part. To clear filter press Cancel;

 

The filter activity is indicated by the green color of the button.

Also you can fast load all server objects. For this click button Load All Server Objects (2), set filters if need (3,4,5) and press Start (7).
